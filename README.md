Für die Durchführung der Aufgabenstellung wurde CSS Grid verwendet. 

## Gründe

1. Keine Extra Installation notwendig:
Css Grid ist direkt im Browser eingebaut, bootstrap müsste extra als Paket installiert werden. 

2. Weniger Code im Html file und Seperation of Concerns
Mit Css Grid bleibt die Layout-Logik im CSS. Bei bootstrap müssten im html viele Klassen hinterlegt werden (zB class="col-md-10")

3. Einfache responsive Anpassung
mit @media queries kann das Layout für verschiedene Bildschirmgrößen einfach geändert werden.
- mobil: Tabellen sind 12 Spalten breit
- desktop: Tabellen sind 10 Spalten breit und zentriert

4. Funktioniert in allen modernen Browsern
Css Grid wird von allen gängigen Browser unterstützt
